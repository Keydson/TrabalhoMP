INTRUÇÕES PARA A COMPILAÇÃO E EXECUÇÃO DO PROGRAMA

Para compilar o programa, entre pelo terminal no diretório "../TrabalhoMP/src", onde está contido o arquivo makefile;

Para compilar o programa, utilize no terminal "$make Velha", e depois "$./Velha" para executá-lo;

Para compilar o arquivo de teste, utilize no terminal "$make Testa_Velha", para executá-lo, utilize "$./Testa_Velha";

Para excluir os arquivos ".o", utilize no terminal "$make clean".
